Part 2

- Use of HTML tags
- Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag (the paragraph tags, or the bold and italics tags are examples of this). Some tags (called "void" tags) do not require a closing tag (like the "break" tag).
- Element: An HTML element refers to everything within a set of opening and closing tags.
- Attribute: This is a property of an HTML element. The href attribute of the "a" element, specifies a link, for example.
